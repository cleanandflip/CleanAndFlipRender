Placeholder Text Color Fix Instructions for Replit
1. GLOBAL PLACEHOLDER FIX
css/* Make ALL placeholder text white across the entire site */
::placeholder {
  color: #ffffff !important;
  opacity: 0.7; /* Slight transparency to differentiate from typed text */
}

/* For better browser compatibility */
::-webkit-input-placeholder { /* Chrome/Opera/Safari */
  color: #ffffff !important;
  opacity: 0.7;
}
::-moz-placeholder { /* Firefox 19+ */
  color: #ffffff !important;
  opacity: 0.7;
}
:-ms-input-placeholder { /* IE 10+ */
  color: #ffffff !important;
  opacity: 0.7;
}
:-moz-placeholder { /* Firefox 18- */
  color: #ffffff !important;
  opacity: 0.7;
}
2. TAILWIND PLACEHOLDER CLASSES
css/* If using Tailwind, update your input components */
input,
textarea,
select {
  @apply placeholder:text-white placeholder:opacity-70;
}

/* Or more specific */
.form-input {
  @apply placeholder:text-white;
}
3. SPECIFIC FORM FIELD UPDATES
css/* Target specific input types */
input[type="text"],
input[type="email"],
input[type="password"],
input[type="tel"],
textarea {
  @apply text-white 
         placeholder:text-white 
         placeholder:opacity-80;
}

/* Make placeholder slightly dimmer than typed text */
input:not(:placeholder-shown) {
  @apply text-white;
}
4. COMPONENT-LEVEL FIX
If using React/component-based approach:
jsx// Update your input component classes
<input
  type="text"
  placeholder="First Name"
  className="... placeholder:text-white placeholder:opacity-80"
/>
5. ALTERNATIVE APPROACHES
css/* Option 1: Very bright placeholders */
::placeholder {
  color: rgba(255, 255, 255, 0.9) !important;
}

/* Option 2: Slightly gray but still visible */
::placeholder {
  color: #e5e7eb !important; /* gray-200 */
}

/* Option 3: Pure white with reduced opacity */
::placeholder {
  color: white !important;
  opacity: 0.6;
}
6. QUICK GLOBAL OVERRIDE
Add this to your main CSS file or global styles:
css/* Force all placeholders to be white */
* {
  &::placeholder {
    color: white !important;
    opacity: 0.75;
  }
}
7. CHECK FOR CONFLICTS
css/* Remove any existing gray placeholder styles */
.placeholder\:text-gray-400::placeholder,
.placeholder\:text-gray-500::placeholder,
.placeholder\:text-gray-600::placeholder,
[class*="placeholder:text-gray"]::placeholder {
  color: white !important;
}
IMPLEMENTATION STEPS:

Add the global ::placeholder rule to your main CSS file
Remove any existing placeholder:text-gray-* classes from inputs
Add placeholder:text-white to your input field classes
Test across different browsers for compatibility

This will make all placeholder text white and clearly visible on the dark background while maintaining good contrast with actual typed text!
