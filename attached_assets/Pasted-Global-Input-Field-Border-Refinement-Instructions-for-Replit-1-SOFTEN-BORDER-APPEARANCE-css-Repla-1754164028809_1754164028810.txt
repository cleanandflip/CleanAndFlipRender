Global Input Field Border Refinement Instructions for Replit
1. SOFTEN BORDER APPEARANCE
css/* Replace sharp borders with softer, more subtle ones */
input,
textarea,
select,
.form-input,
[type="text"],
[type="email"],
[type="password"],
[type="tel"],
[type="number"] {
  @apply border border-gray-700/40 /* Lower opacity */
         bg-gray-800/50 /* Semi-transparent background */
         backdrop-blur-sm
         rounded-lg /* Larger border radius */
         transition-all duration-300 ease-out /* Smooth transitions */
         shadow-inner shadow-black/20; /* Subtle inner shadow */
}

/* Hover state - gentle transition */
input:hover,
textarea:hover,
select:hover {
  @apply border-gray-600/50 
         bg-gray-800/70;
}

/* Focus state - smooth glow instead of sharp border */
input:focus,
textarea:focus,
select:focus {
  @apply border-gray-500/50 
         bg-gray-800/80
         outline-none
         ring-2 ring-gray-600/20 /* Very subtle ring */
         ring-offset-0;
}
2. ADD PAGE LOAD FADE-IN ANIMATION
css/* Prevent jarring appearance on load */
@keyframes fadeInForm {
  from {
    opacity: 0;
    transform: translateY(10px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* Apply to form elements */
form,
.form-container {
  animation: fadeInForm 0.4s ease-out;

/* Stagger input appearances */

4. PREVENT FLASH OF UNSTYLED CONTENT (FOUC)
css/* Add to the very top of your global CSS */
html {
  visibility: hidden;
  opacity: 0;
}

html.loaded {
  visibility: visible;
  opacity: 1;
  transition: opacity 0.3s ease-out;
}
Add this JavaScript:
javascript// Add to your app initialization
document.addEventListener('DOMContentLoaded', () => {
  document.documentElement.classList.add('loaded');
});

css/* Complete replacement for sharp borders */
:root {
