Fix Dropdown Z-Index and Screen Jumping Issues
Instructions for Replit:
"The dropdown menus are appearing BEHIND cards and the screen jumps when clicking buttons. This is a stacking context problem. Fix these issues:
1. Find What's Creating Stacking Contexts
Run these commands to find the culprits:
bash# Find all transform properties (these create stacking contexts)
grep -rn "transform:\|translate\|scale\|rotate" --include="*.css" --include="*.tsx" client/src/

# Find all filter properties (these create stacking contexts)
grep -rn "filter:\|backdrop-filter:" --include="*.css" --include="*.tsx" client/src/

# Find position relative/fixed on card containers
grep -rn "relative\|fixed\|sticky" --include="*.tsx" client/src/components/

# Find will-change properties
grep -rn "will-change" --include="*.css" --include="*.tsx" client/src/
2. Remove Stacking Context Creators from Parent Elements
The cards or their containers likely have these properties that need removal:
css/* REMOVE these from card/table/container elements: */
.card, .table-container, main, section {
  /* transform: none !important; */
  /* filter: none !important; */
  /* backdrop-filter: none !important; */
  /* will-change: auto !important; */
}
3. Fix the Export/Actions Dropdown
Find the dropdown component and ensure it:
typescript// Use a portal to render outside the stacking context
import { createPortal } from 'react-dom';

// Or use Radix UI's Popover/DropdownMenu with proper z-index
<DropdownMenu.Content 
  className="z-[9999]"
  sideOffset={5}
  align="end"
  style={{ zIndex: 9999 }}
>
4. Fix Screen Jumping on Button Click
The jumping is likely from:

Focus styles adding borders/outlines
Active states changing dimensions
Transform scale animations

Fix by:
css/* Ensure buttons don't change size on interaction */
button {
  /* Prevent layout shift */
  box-sizing: border-box;
  outline-offset: -2px; /* Keep outline inside */
}

/* Use transform for animations, not size changes */
button:active {
  transform: scale(0.98); /* Don't change actual dimensions */
}
5. Update Z-Index Hierarchy
Create a proper z-index system:
css:root {
  --z-dropdown: 50;
  --z-modal: 100;
  --z-popover: 200;
  --z-tooltip: 300;
  --z-notification: 400;
  --z-command-palette: 500;
  --z-max: 9999;
}

/* Apply to dropdowns */
.dropdown-content,
[role="menu"],
[data-radix-popper-content-wrapper] {
  z-index: var(--z-max) !important;
}
