2. Single Clear Button
Right now there are two “X”s:

One is your native browser clear button for <input type="search">

One is your custom “clear search” button.

Fix: hide the browser’s native clear icon using CSS:

css
Copy
Edit
/* Hide default search input clear icon */
input[type="search"]::-webkit-search-decoration,
input[type="search"]::-webkit-search-cancel-button,
input[type="search"]::-webkit-search-results-button,
input[type="search"]::-webkit-search-results-decoration {
  display: none;
}
Put this in your global.css or header component styles.

3. Spinner + Clear Button Alignment
Only show one element on the right at a time:
Spinner if loading, otherwise Clear (X) if there’s a search term.
